# ✅ Версии 2.4.1 — 2.4.2 (20.01.2026) - Поиск и выгрузка в VK Mini App

Это крупное обновление, сфокусированное на значительном расширении функциональности и повышении надежности VK Mini App.

## 🚀 Новые возможности

*   **🔎 Поиск и выгрузка файлов прямо в Mini App:**
    *   **Интерфейс:** В мини-приложение добавлена полноценная **модальная панель поиска**, которая позволяет находить файлы в облаке, не покидая интерфейс ВКонтакте.
    *   **Бэкенд:** Созданы новые API эндпоинты (`/api/search` и `/api/download`) для выполнения поиска по базе данных D1 и последующей потоковой выгрузки найденных файлов напрямую из облака в браузер пользователя.
    *   **Результаты:** Найденные файлы отображаются в виде удобного списка с иконками, датами и папками хранения.

## 🔧 Исправления и улучшения

*   **🛡️ Отказоустойчивость загрузчика в Mini App (v2.4.1):**
    *   **Проблема:** Ранее, если файл успешно загружался в облако, но происходил сбой при записи метаданных в базу, файл мог "потеряться".
    *   **Решение:** Теперь в таком случае загрузка переходит в статус **"⚠️ Ошибка базы!"** и появляется кнопка **"Повторить"**. Она позволяет перезапустить только операцию записи в базу, без необходимости заново загружать сам файл.

*   **↩️ Улучшенная отмена загрузок (v2.4.1):**
    *   При отмене загрузки теперь появляется кнопка **"Вернуть"**, которая позволяет восстановить файл обратно в очередь, если отмена была случайной.

*   **⚙️ Корректная работа с папками Google Drive (v2.4.2):**
    *   **Проблема:** Хранение только имени папки могло приводить к ошибкам, если у пользователя было несколько папок с одинаковыми именами.
    *   **Решение:** Теперь для Google Drive в базе данных хранится и имя, и уникальный ID папки (`FolderName|folderId`), что гарантирует корректную работу с файлами.
