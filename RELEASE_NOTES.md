# ✅ Версия 2.4.6 (27.01.2026) - Чат с ИИ и критические исправления

Это кумулятивное обновление, которое включает в себя как новые функции, так и важные исправления из пропущенной версии 2.4.5. Основной фокус — добавление чата с ИИ в VK Mini App и решение критических проблем с загрузкой файлов.

## 🚀 Новые возможности

*   **⭐ Чат с ИИ в VK Mini App:**
    *   Прямо в интерфейс приложения добавлен полноценный чат с нейросетью. Теперь пользователи могут задавать вопросы и получать ответы от AI, не покидая VK.

## 🔧 Критические исправления

*   **⭐ Исправлена загрузка в Dropbox (v2.4.5):**
    *   **Проблема:** Файлы с не-латинскими (например, русскими) именами не загружались в Dropbox из-за ошибки кодирования.
    *   **Решение:** Реализовано корректное экранирование символов в именах файлов. **Загрузка в Dropbox теперь работает стабильно.**

*   **⭐ Исправлено сохранение папки для WebDAV (v2.4.6):**
    *   **Проблема:** При настройке WebDAV-подключения через Mini App, указанная пользователем папка для загрузки не сохранялась.
    *   **Решение:** Ошибка полностью исправлена. Настройки папки теперь корректно сохраняются на бэкенде.

## ⚙️ Улучшения и оптимизация

*   **🚀 Ускоренная загрузка в WebDAV (v2.4.6):
    *
    *   Загрузчик для WebDAV-совместимых облаков (включая Mail.ru) был полностью переработан. Использование потоковой передачи (streaming) делает загрузку значительно быстрее и надежнее.

*   **Улучшенная форма подключения WebDAV (v2.4.5):**
    *   В интерфейс Mini App добавлено поле "Папка для сохранения", что позволяет пользователям сразу указать нужную директорию при настройке.

*   **Стабилизация загрузки в Google Drive (v2.4.6):**
    *   Скорректированы CORS-заголовки для `PUT` запросов, что повышает надежность возобновляемой загрузки файлов в Google Drive.
