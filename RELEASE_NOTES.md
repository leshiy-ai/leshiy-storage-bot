## Что нового в v2.3.2

Этот релиз добавляет интеграцию с ВКонтакте, улучшенную реферальную систему и оптимизированный AI-поиск.

### 🚀 Новые возможности и улучшения

*   **Интеграция с ВКонтакте:** Бот теперь может работать в сообществах VK, обрабатывая вебхуки, принимая файлы (включая мультизагрузку), выполняя команды и запуская AI-анализ.
*   **Улучшенная реферальная система:** Ссылки-приглашения, создаваемые командой `/share`, теперь привязываются к конкретной папке.
*   **Оптимизация AI-поиска:** Процесс интеллектуального поиска был улучшен. Бот сначала отбирает кандидатов с помощью SQL-запроса, а затем отправляет короткий список нейросети для анализа.
*   **Новые способы подключения:** Добавлены меню для подключения к WebDAV-серверам, включая инструкцию для Облака Mail.ru.
*   **Доработанная выгрузка файлов:** Улучшен механизм скачивания файлов из облака по кнопке "Выгрузить".

### 🔧 Внутренние улучшения

*   Добавлены новые переменные окружения для работы с VK и Mail.ru OAuth: `VK_GROUP_TOKEN`, `VK_GROUP_ID`, `VK_TOKEN`, `MAILRU_CLIENT_ID`, `MAILRU_CLIENT_SECRET`.
