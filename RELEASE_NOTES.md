# 🔥 Версия 2.4.8 (28.01.2026) - Интерактивная реферальная система

Это обновление полностью преображает механику приглашений в VK Mini App, делая её интуитивной, визуально привлекательной и "вирусной".

## 🚀 Новые возможности

*   **⭐ Интерактивный баннер приглашения:**
    *   Когда пользователь переходит в приложение по реферальной ссылке, его встречает красивый баннер с **именем и аватаром пригласившего друга**.
    *   Процесс подключения к хранилищу друга теперь занимает **один клик** по кнопке на баннере.

## ⚙️ Улучшения и UX

*   **✨ Свайп для отказа:** На мобильных устройствах баннер можно просто "смахнуть" с экрана, если приглашение неактуально.
*   **🔗 Умная обработка ссылок:** Переработан механизм парсинга реферальных ссылок. Теперь бот корректно распознает ID друга из любых форматов (прямые ссылки VK, ссылки из Telegram, QR-коды).
*   **🎨 Плавные анимации:** Появление, скрытие и свайп баннера сопровождаются плавными анимациями, что делает интерфейс живым и отзывчивым.
