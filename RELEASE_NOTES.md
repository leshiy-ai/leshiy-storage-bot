## Что нового в v2.3.0

Этот релиз представляет две мощные AI-функции: анализ документов и глубокий анализ видео. Также обновлены AI-модели для повышения производительности.

### 🚀 Новые возможности и улучшения

*   **📄 Анализ документов (AI):**
    *   Теперь бот умеет "читать" и анализировать содержимое документов (PDF, DOCX и др.).
    *   При загрузке документа AI автоматически создает краткое описание его содержимого, которое сохраняется в базе данных. Это позволяет в будущем легко находить нужные файлы по их сути, а не только по имени.

*   **🎬 Глубокий анализ видео (AI):**
    *   Функция анализа видео была кардинально улучшена. Вместо простого транскрибирования аудиодорожки, новая система предоставляет полный покадровый разбор видео.
    *   AI описывает ключевые действия, объекты, сцены и распознает речь, создавая подробный текстовый отчет о содержании видеофайла.

*   **⚡️ Оптимизация AI-моделей:**
    *   Большинство сервисов, использующих API Gemini, переведены на более быструю и экономичную модель `gemini-2.5-flash-lite`.
    *   Это изменение обеспечивает более быстрый отклик от AI-функций, таких как создание описаний для фото, видео и документов.

### 🔧 Внутренние улучшения

*   Код был адаптирован для интеграции новых мультимодальных возможностей Gemini API.
*   Добавлена поддержка новых MIME-типов для корректной обработки различных форматов документов.
