## Что нового в v2.2.0

Это крупное обновление, которое выводит бота на новый уровень. Мы добавили продвинутые возможности искусственного интеллекта, улучшили управление файлами и расширили функциональность для администраторов и пользователей.

### 🚀 Новые функции

*   **🧠 Продвинутая интеграция с ИИ:**
    *   **Панель управления ИИ:** Новая команда `/admin` (и ее псевдоним `/ai_settings`) открывает доступ к меню, где можно гибко настраивать, какие AI-модели использовать для разных задач.
    *   **Мульти-провайдерность:** Добавлена поддержка нескольких поставщиков ИИ: Cloudflare Workers AI (бесплатно), Google Gemini и BotHub. Вы можете переключаться между ними, чтобы выбрать оптимальное решение.
    *   **Анализ документов:** Бот теперь может «читать» и описывать содержимое документов (PDF, DOCX и т.д.), которые вы ему отправляете. Эта функция работает в фоновом режиме.
    *   **Анализ видео:** Реализована функция анализа видео. Бот может описать, что происходит в видеоролике, и распознать речь на аудиодорожке.

*   **📂 Улучшенное управление файлами:**
    *   **Интерактивный поиск:** Теперь команда `/search` работает в два шага. Сначала вы отправляете команду, а затем — поисковый запрос, что делает процесс более интуитивным.
    *   **Постраничные результаты поиска:** Результаты поиска выводятся страницами по 5 файлов, с удобной навигацией.
    *   **Статус доступности файла:** В результатах поиска теперь отображается, доступен ли файл для выгрузки с вашим текущим подключением.
    *   **Выгрузка файлов из облака:** Новая функция позволяет скачивать файлы из вашего облачного хранилища прямо в чат Telegram.

*   **🤝 Улучшенная реферальная система:**
    *   **Привязка к папке:** Теперь при создании реферальной ссылки (`/share`) она автоматически привязывается к той папке, которая у вас выбрана в данный момент. Друзья, перешедшие по ссылке, будут загружать файлы именно в эту папку.

### ✨ Улучшения

*   **🤖 Более точное определение типов файлов:** Улучшен алгоритм определения типа загружаемого файла (фото, видео, аудио, документ).
*   **🔐 Повышенная безопасность WebDAV:** Сообщения, содержащие URL для подключения к WebDAV с логином и паролем, автоматически удаляются из чата после настройки.
*   **🗄️ Хранение метаданных:** Информация о файлах, включая сгенерированные ИИ описания, теперь сохраняется в базе данных для более быстрого доступа и поиска.
*   **⚡️ Фоновая обработка:** Генерация описаний для файлов с помощью ИИ теперь происходит в фоновом режиме, что делает работу бота быстрее и отзывчивее.
