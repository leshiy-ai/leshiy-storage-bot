# ✅ Версия 2.4.0 (17.01.2026) - Прямая загрузка из VK Mini App и AI-поиск

## 🚀 Новые возможности

*   **⚡️ Прямая потоковая загрузка из VK Mini App (Client-to-Cloud):**
    *   **Архитектура:** Механизм загрузки файлов из мини-приложения полностью переработан. Теперь файлы отправляются из вашего браузера напрямую в облачное хранилище (Яндекс.Диск, Google Drive и др.), минуя сервер Cloudflare.
    *   **Снятие ограничений:** Это полностью снимает ограничение Cloudflare на размер загружаемого файла. Теперь вы можете загружать **файлы любого размера**.
    *   **Скорость и UX:** Загрузка происходит значительно быстрее. В интерфейсе Mini App появился **прогресс-бар**, который в реальном времени показывает статус загрузки каждого файла.
    *   **Управление:** Добавлена возможность отмены загрузки и управления очередью файлов.

*   **🧠 Улучшенный поиск для администраторов:**
    *   Команда `/search` теперь поддерживает **интеллектуальный поиск**. Если запрос администратора содержит пробел (например, `/search фото с пляжа`), бот автоматически запускает AI-поиск по содержимому и описаниям файлов, а не только по их именам.

## 🔧 Исправления и улучшения

*   **☁️ Отказ от OAuth для Mail.ru:** Удален устаревший и нестабильный метод подключения к Облаку Mail.ru через OAuth. Теперь подключение происходит исключительно через **WebDAV**, что делает настройку проще и надежнее.

## 🛠️ Для разработчиков

*   **Новые API эндпоинты:** Для реализации прямой загрузки были добавлены новые маршруты:
    *   `POST /api/get-upload-link`: Получает от бэкенда временную ссылку для прямой загрузки файла в облако.
    *   `POST /api/confirm-upload`: Сообщает бэкенду об успешной загрузке для сохранения метаданных в базу D1.
*   **Потоковая передача:** Функции загрузки в облака (`uploadTo...`) были переписаны для работы с `ReadableStream` (`uploadTo...Stream`), что обеспечивает передачу данных "на лету" без накопления в памяти воркера.
