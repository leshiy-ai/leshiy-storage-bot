## Что нового в v2.2.2

Этот релиз сфокусирован на значительном улучшении и оптимизации функции поиска, а также на внутреннем рефакторинге кода для работы с искусственным интеллектом.

### 🚀 Новые возможности и улучшения

*   **⚡️ Умный и быстрый поиск:**
    *   **Два режима поиска:** Теперь поиск работает в двух режимах. Для простых запросов (одно слово, часть имени) используется быстрый поиск по базе данных. Для сложных, "человеческих" запросов (например, "видео с собакой на пляже") автоматически включается интеллектуальный AI-поиск.
    *   **Оптимизация AI-поиска:** Процесс интеллектуального поиска был полностью переработан. Бот сначала отбирает наиболее вероятных кандидатов с помощью быстрого SQL-запроса, и только затем отправляет этот короткий список нейросети для финального анализа. Это делает AI-поиск значительно быстрее и эффективнее.
    *   **Расширенные права для администраторов:** Администраторы теперь могут выполнять обычный поиск по файлам *всех* пользователей, а не только по своим.

*   **🔧 Внутренние улучшения:**
    *   **Рефакторинг AI-модуля:** Код для взаимодействия с AI-моделями (Gemini, Workers AI) был унифицирован. Это упрощает добавление новых моделей и поддержку существующих, а также повышает общую стабильность системы.
    *   **Оптимизация моделей:** Проведены тесты с использованием более легковесных и быстрых моделей, таких как `gemini-2.5-flash-lite`, для ускорения ответов бота.

### 🐞 Исправления

*   Повышена точность и релевантность результатов интеллектуального поиска.
*   Улучшена стабильность при обработке сложных поисковых запросов.
