## Что нового в v2.3.3

Этот релиз представляет масштабное обновление для ВКонтакте, добавляет гибкое управление AI-моделями и улучшает стабильность работы с облаками.

### 🚀 Новые возможности и улучшения

*   **Прокачанная интеграция с VK:**
    *   **Интерактивность:** VK-бот теперь поддерживает инлайн-кнопки и полноценные диалоговые сценарии, как в Telegram.
    *   **Мультизагрузка:** Загружайте несколько файлов одним сообщением — бот обработает все.
    *   **Умный парсер видео:** Новый механизм пытается "достать" прямую ссылку на видео, даже если VK API ее не отдает.
    *   **"Ультимативный перехватчик":** Бот научился обходить страницы-заглушки VK, находя прямые ссылки на документы внутри HTML.
    *   **AI-анализ для VK:** Фоновое описание контента (фото, видео, документы) теперь доступно и для файлов из ВКонтакте.
*   **Гибкое меню настроек AI (`/ai_settings`):**
    *   Администраторы теперь могут "на лету" переключать AI-модели для разных задач (чат, анализ изображений, распознавание речи и др.) через новое интерактивное меню.
*   **Улучшения WebDAV:**
    *   Более надежный парсинг ссылок для подключения.
    *   Создание папок теперь происходит по стандартному протоколу (`MKCOL`).
*   **Оптимизация выгрузки:**
    *   Улучшен процесс скачивания файлов из облачных хранилищ, особенно для Dropbox.
*   **Ускоренный AI-поиск:**
    *   Логика интеллектуального поиска оптимизирована за счет более быстрых SQL-запросов, что делает его отзывчивее.

### 🔧 Внутренние улучшения

*   Код был реорганизован для лучшей поддержки мульти-платформенной логики (Telegram и VK).
*   Обновлены и унифицированы функции вызова AI-моделей.
