# 🔥 Версия 2.5.0 (29.01.2026) - Интерактивная система приглашений

Это обновление кардинально меняет механику "Поделиться" в VK Mini App, делая ее интуитивно понятной, визуально привлекательной и способствующей "вирусному" распространению.

## 🚀 Новые возможности

*   **⭐ Интерактивная панель "Поделиться":**
    *   При вызове команды `/share` теперь появляется элегантная модальная панель.
    *   **Предпросмотр:** Панель наглядно показывает, к какому облаку и папке предоставляется доступ, а также отображает сгенерированный код приглашения.
    *   **Нативная интеграция:** Кнопка "Отправить" использует `VKWebAppShare`, вызывая стандартный диалог "Поделиться" ВКонтакте для отправки красивого сообщения с кнопкой-приглашением.

## ⚙️ Улучшения и UX

*   **🔗 Улучшенный API для инвайтов:**
    *   Добавлены новые эндпоинты (`/api/create-invite`, `/api/get-invite-info`) для генерации и проверки кодов приглашений.
    *   Эндпоинт `/api/connect-friend` стал универсальным и теперь принимает как ID пользователя, так и новые коды приглашений.
*   **📱 Свайпаемые панели:** Все информационные панели в Mini App (Поделиться, Админ-меню, Отладка) теперь можно интуитивно закрывать свайпом на мобильных устройствах.
*   **📋 Удобное копирование:** На панель добавлены кнопки для быстрого копирования ссылки и текста приглашения в буфер обмена.
