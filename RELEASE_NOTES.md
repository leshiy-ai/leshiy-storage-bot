# ✅ Версия 2.3.8 (15.01.2026) - Интеграция с VK Mini App и улучшения

## 🚀 Новые возможности

*   **📱 VK Mini App:**
    *   **Полноценный интерфейс:** Теперь ботом можно управлять через удобное мини-приложение внутри ВКонтакте.
    *   **Управление хранилищем:** Просматривайте статус подключения, свободное место на диске и текущую папку для загрузки.
    *   **Интерактивный выбор папок:** Просматривайте, выбирайте и создавайте папки в облаке прямо из интерфейса VK.
    *   **Настройки AI:** Администраторы могут гибко настраивать AI-модели для разных задач.
    *   **Современный UI/UX:** Интерфейс поддерживает обновление "потягиванием" (Pull-to-refresh) и использует модальные окна для удобной навигации.

*   **☁️ Улучшения WebDAV:**
    *   Добавлена поддержка просмотра списка папок для WebDAV-хранилищ.

*   **⚡️ Оптимизация загрузки из VK:**
    *   Переход на потоковую загрузку файлов (`ReadableStream`) из VK, что значительно ускоряет процесс и снижает нагрузку.

*   **🧠 Обновление AI:**
    *   Модель для текстового чата обновлена до `gemini-2.5-flash` для еще более качественных и быстрых ответов.

## 🔧 Исправления и улучшения

*   **Dropbox & Google Drive:** Улучшена и стабилизирована функция получения списка папок.
*   **Обработка вложений VK:** Бот теперь корректно обрабатывает файлы из пересланных и отвеченных сообщений.
*   **Авторизация:** Улучшен процесс авторизации в VK для мобильных браузеров.
