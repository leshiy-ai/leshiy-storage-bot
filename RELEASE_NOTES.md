# 🔥 Версия 2.5.1 (30.01.2026) - Тёмная тема, Языки и UX

Это обновление фокусируется на качестве пользовательского опыта, добавляя долгожданную поддержку тем оформления, зачатки локализации и множество мелких, но важных улучшений в интерфейсе и логике.

## 🚀 Новые возможности

*   **🎨 Тёмная и светлая темы:**
    *   Приложение теперь **автоматически адаптируется** под системную тему ВКонтакте (светлую или тёмную).
    *   Добавлена **кнопка (☀️/🌙) для ручного переключения** темы. Выбор пользователя сохраняется.

*   **🌐 Интернационализация (i18n):**
    *   Внедрена базовая инфраструктура для поддержки **русского и английского языков**.
    *   В шапку добавлена кнопка-флаг (🇷🇺/🇺🇸) для быстрой смены языка.

## ✨ Улучшения и UX

*   **⚙️ Надёжное подключение друга:** Существенно переработан эндпоинт `/api/connect-friend`. Теперь при подключении по инвайту права доступа и токены владельца **полностью клонируются**, обеспечивая стабильную и корректную работу "общего диска".
*   **💅 Плавное подключение WebDAV:** Процесс настройки WebDAV стал асинхронным, панель больше не "зависает" в ожидании ответа.
*   **🔍 Адаптивные бейджи в поиске:** Индикаторы статуса файлов в результатах поиска теперь корректно отображаются в обеих темах.
*   **💬 Иконки в Telegram:** Ответы команд `/share`, `/debug`, `/disconnect` в Telegram-боте были обогащены иконками для лучшей читаемости.
*   **🐞 Исправлена логика создания папок** в модальном окне, убраны визуальные "моргания".
