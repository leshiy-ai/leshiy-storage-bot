# Релиз v3.0.5

### Новые возможности 🎉
**Поддержка Yandex.Cloud:** Добавлена нативная интеграция с YandexGPT для чата и Yandex SpeechKit для распознавания речи.

### Улучшения и исправления 🛠️
*   **Улучшен UX:** После подключения нового облачного хранилища (Яндекс, Google, Mail.ru) пользователю теперь автоматически предлагается выбрать папку для загрузки.
*   **Рефакторинг:** Оптимизирован код для работы с FTP/SFTP и получения списка папок.
