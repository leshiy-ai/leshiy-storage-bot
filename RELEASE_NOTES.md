# 🚀 Релиз v2.1.5: Умный поиск с AI и новая архитектура

Это масштабное обновление превращает "Хранилку" из простого файлового менеджера в интеллектуального медиа-архивариуса. Главное нововведение — глубокая интеграция AI для семантического поиска и автоматического анализа контента.

## ✨ Ключевые нововведения

*   **🧠 Семантический поиск по файлам (`/search`)**: Новая команда, которая позволяет искать фото и видео по их содержанию, а не только по имени. Просто опишите, что вы ищете (например, `/search документы за прошлый месяц` или `/search кот на диване`), и AI найдет соответствующие файлы.
*   **👁️ Автоматическое описание контента**: Бот теперь использует AI-модель (Google Gemini) для "просмотра" загружаемых изображений и видео, создавая для них подробные текстовые описания. Эти описания служат основой для умного поиска.
*   **💬 Разговорный AI**: Бот стал умнее и теперь может поддерживать более естественный диалог, отвечая на ваши вопросы о его функциях.
*   **⚙️ Панель управления AI для администратора (`/ai_settings`)**: Администраторы теперь могут выбирать, какие AI-модели использовать для чата, анализа изображений и создания векторных представлений (embeddings) для поиска.

## 🏗️ Архитектурные изменения

*   **⚠️ Breaking Change**: Для хранения метаданных файлов и их AI-описаний теперь используется база данных **Cloudflare D1**. Это было необходимо для реализации быстрого и сложного поиска. **KV Namespace** по-прежнему используется для хранения токенов и настроек пользователей.
*   **Обновление `wrangler.toml`**: В конфигурационный файл добавлена обязательная привязка (binding) для базы данных D1 с именем `FILES_DB` и для Workers AI c именем `AI`.
*   **Новая переменная окружения**: Для работы AI-функций теперь необходим секрет `GEMINI_API_KEY`.

## 🛠️ Исправления и улучшения

*   Код был значительно переработан для интеграции с D1 и AI-сервисами.
*   Документация (`README.md`) полностью переписана, чтобы отразить новые функции и шаги по настройке.

## Что дальше?

Это только первый шаг в интеграции AI. В будущих версиях планируется расширить возможности поиска, добавить поддержку большего числа форматов файлов и улучшить диалоговые навыки бота.
